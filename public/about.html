About This Site

This is a really simple image gallery, intended as more than a "hello world" example' to be implemented using different MVC frameworks, so they can be compared.

This site started out as the "Simple Gallery" from Bryant Smith.

The Images

Images are stored in a "data" folder inside the document root, and they are described in an "images" table, for which a SQL script is provided.

Supporting Technologies

The website uses the jquery Javascript framework, so that it can also use the jquery lightbox plugin.

The Story

This website was created for the Spring 2014 session of ACIT2910, at BCIT.
